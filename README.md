
## Features
- Complete EDA with visualizations
- Professional data preprocessing and feature engineering
- Multiple ML algorithms with hyperparameter tuning
- Cross-validation and model evaluation
- Custom prediction functionality
- Web interface for interactive predictions

## Files
- `titanic_ml_project.py`: Main ML pipeline implementation
- `app.py`: Flask web application
- `run_ml_project.py`: Script to run the ML pipeline
- `run_website.py`: Script to run the web interface
- `requirements.txt`: Dependencies
- `data/`: Directory for datasets
